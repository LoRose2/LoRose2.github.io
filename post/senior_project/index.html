Actuarial Risk Models

By Logan Rosentreter

September 1, 2022

Modeling the aggregate claim amount where each claim amount is a random variable and the number of claims is also a random variable is important in actuarial science. We present a general class of distributions whose characteristics can be used to develop an ad-hoc procedure for determining which distribution is appropriate for the number of claims based on observed data. If the claim amounts take on only positive integer values, we derive the distribution of the aggregate claim amount.

Suppose a city gets ravaged by a tornado. Throughout the city, there will be a random number of houses hit by the tornado which will result in a claim, and each claim will have a random amount from the damages caused. For actuaries, it is important to be able to model the aggregate claim amount. An actuarial risk model or loss model is a concise mathematical description of the behavior of a collection of risks that make up an insurance portfolio. One way to model the number of the claims would be to use the binomial distribution, which is a probability distribution that models the number of successes out of a number of specified independent trials. It is also possible to use other discrete distributions such as the Poisson distribution or the negative binomial distribution. We will consider a general class of distributions that will provide an ad-hoc procedure for choosing a discrete distribution that is appropriate for the data on hand. Let $X_i$ be the random claim amount for the $i^{th}$ claim and let $N$ be the random number of claims. The aggregate claim amount can be written as $S = X_1 + X_2 + \ldots + X_N$. In order to find the distribution of the aggregate claim amount, the distributions of the individual random claim amount and the random number of claims need to be determined. We will derive the distribution of the aggregate claim amount $S$ based on the distributions of the claim amounts $X_i, i = 1, 2, \ldots, N$, and the number of claims $N$.
